# 솔루션 확장 및 고도화 로드맵

## Executive Summary

본 문서는 그린수소 프로젝트 Bankability 분석 에이전트의 중장기 확장 및 고도화 계획을 정의합니다. 현재 MVP 수준의 핵심 기능을 바탕으로, 단계적으로 기능을 확장하고 새로운 시장과 도메인으로 진출하는 전략을 제시합니다.

**보고서 유형**:

| 유형 | 대상 | 페이지 | 내용 |
|------|------|--------|------|
| Executive Summary | 경영진/이사회 | 2-3p | 핵심 KPI, 투자 권고 |
| Investment Memo | 금융기관 심사역 | 10-15p | Bankability 분석, 리스크 평가 |
| Technical Report | 기술 실사팀 | 20-30p | 설비 상세, 운영 계획 |
| Due Diligence Pack | 법무/회계 자문사 | 50p+ | 전체 분석 데이터 |

**기술 스택**:
- PDF 생성: `@react-pdf/renderer` 또는 Puppeteer
- 템플릿 엔진: Handlebars
- 차트 내보내기: Recharts to SVG/PNG
- AI 요약: Claude API를 활용한 섹션별 자동 서술

**보고서 구조 예시**:
```
1. Executive Summary (AI 자동 생성)
2. Project Overview
   - 사업 개요
   - 기술 구성
   - 입지 분석
3. Financial Analysis
   - 투자비 구조 (CAPEX Breakdown)
   - 운영비 분석 (OPEX Structure)
   - 현금흐름 전망
4. Bankability Assessment
   - DSCR 분석 (P50/P90/P99)
   - 민감도 분석
   - 스트레스 테스트 결과
5. Risk Analysis
   - 주요 리스크 요인
   - 완화 방안
6. Recommendations
   - 최적 금융 구조
   - 인센티브 활용 전략
7. Appendix
   - 상세 가정
   - 시간별 운영 데이터
```

    org_id UUID REFERENCES organizations(id),

);

);
